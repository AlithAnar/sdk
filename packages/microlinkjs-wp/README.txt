Render beautiful link previews with simple shortcodes.

== Description ==

Display previews via the Microlink API with easy-to-use `[microlink/]` shortcodes.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `microlinkjs.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. If required, set API key via **Settings** -> _Microlinkjs_
